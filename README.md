# 🛡️ KALI LINUX TO GIT-HUB

তোর কালির টার্মিনাল থেকে ফাইল তৈরি করে গিটহাবে পুশ করার সম্পূর্ণ গাইডলাইন নিচে দেওয়া হলো।

---

### 📝 ১. ফাইল তৈরি ও নোট লেখার নিয়ম

| কিবোর্ড শর্টকাট | কাজ |
| :--- | :--- |
| `nano README.md` | নতুন ফাইল খোলা বা এডিট করা |
| **নোট লিখবি** | তোর প্রয়োজনীয় কমান্ড বা টেক্সট টাইপ করবি |
| `Ctrl + O` তারপর `Enter` | ফাইলটি সেভ (Save) করা |
| `Ctrl + X` | ফাইল থেকে বের (Exit) হওয়া |

---
